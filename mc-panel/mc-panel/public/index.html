How to set up Playit.gg:
1. Sign up at playit.gg and create an agent.
2. Copy the secret key from the agent page.
3. Paste it above and click Save.
4. Click Start — the panel launches a Docker container with the Playit agent.
5. In the Playit dashboard, create a Minecraft Bedrock tunnel → UDP → 127.0.0.1:19132.
6. Share the public IP:port with your friends. 🎉
